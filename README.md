# BBNI Stock Analysis Dashboard

## Objective
Menganalisis data historis saham BBNI 2019-2026 menggunakan Python dan Looker Studio.

## Tools
- Python
- Pandas
- Matplotlib
- Numpy
- Google Sheets
- Looker Studio

## Analysis
- Data Cleaning
- Exploratory Data Analysis
- Trend Analysis
- Volume Analysis
- Dashboard Visualization

## Key Findings
- Harga tertinggi
- Harga terendah
- Rata-rata harga
- Volume perdagangan
